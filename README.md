# Swift Radio app made with SwiftUI

- Listen and save your favorite radio station

## Built with
- Swift
- SwiftUI
- Observable macro
- SPM
- SwiftData

## Requirements
1. Xcode 15.0+

### Features:
- Save and persist favorite station 
- Show information of current show if is available
